Web Crawler

A web crawler created for my algorithms final project. This was completed with a team member. Traces all links accessible to a user specified depth from an inputed website(s) and creates an interactive graph.

Java • Algorithms • Graphs

×

About the project

In collaboration with a classmate for my Algorithms class, we developed a sophisticated web crawler as our final project. The program operates by accepting a file containing links to scrape for 'hrefs', along with a specified depth provided by the user. It meticulously traverses through each link until the designated depth is attained, starting from the initial links. Upon completion, the program generates an interactive graph utilizing a powerful Java graph library. Furthermore, it calculates the Closeness Centrality for each node, presenting the results in descending order. This project not only showcased our adeptness in algorithmic design but also demonstrated our ability to create impactful solutions for real-world challenges.

Maze Solver

Another project that was done for my Data Structures class. I implemented the code that runs through a 2D maze to find the correct solution if one exists.

C++ • Data Structures • Docker

×

About the project

The program begins by reading a 2D maze file and converts it into a matrix format. It then traverses the maze from a predefined starting point, diligently exploring every possible path to find the correct route that leads to the maze's exit, confirming its solvability. Throughout its execution, the program visually illustrates its journey through the maze, pausing to highlight the final, correct path once it's identified.

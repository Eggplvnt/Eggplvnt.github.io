Game of Life

Language(s): C++

Tool(s): CLion

Recreating John Conway's famous Game of Life. A user can either choose a random board to start with, pick a specific one, or even create their own by importing a file. In this program, I use a 2 dimensional vector with life objects. The program goes through an old board, and creates a new board with the correct criteria, determining if a spot should live or die. It's an interesting visual, that is infinite, displaying beautiful and unique patterns.
